# BHInfiniteScrollView



# CocoaPods

1. Add `pod 'BHInfiniteScrollView'` to your Podfile.
2. Run `pod install` or `pod update`.
3. Import \<BHInfiniteScrollView/BHInfiniteScrollView.h\>.

# Requirements

This library requires `iOS 6.0+` and `Xcode 7.0+`.
